# Project Kairo - AI Powered Patient Triage

### Hackathon Project

Kairo is an intelligent patient triage system developed for a hackathon. It leverages AI to rapidly assess patient risk, prioritize care, and assist clinical decision-making with explainable insights.

##  Key Features
- **AI Triage Engine**: Automatically classifies patients into risk categories (Low, Medium, High).
- **Clinical Reasoning**: Provides transparent insights for every recommendation.
- **Modern Dashboard**: High-performance interface built with Next.js and DaisyUI.
- **Department Routing**: Intelligent suggestions for medical department allocation.

##  Built With
- **Next.js 15+**
- **Tailwind CSS & DaisyUI**
- **TypeScript**
- **Bun**
